# v1.1.6 饰品代码审计与修复记录

日期：2026-07-18
分支：`feature/machine-gui-v1.1.6`
状态：自动化门禁通过，游戏内佩戴交互仍需人工验收

## 修复内容

- 统一饰品资源消耗边界，空堆叠不再被识别为可用物品，避免生物脉冲腰带和生命支持箱无限免费消耗。
- 生物脉冲腰带消耗净水瓶后返还玻璃瓶；背包没有空位时将玻璃瓶掉落到玩家身边。
- 生命支持箱的治疗冷却改为玩家级原版物品冷却，替换饰品不再绕过冷却；死亡玩家不会触发治疗或消耗资源。
- 生物脉冲腰带仅在实际工作周期查询 Curios；生命支持箱仅在副作用消减周期或应急治疗可触发时查询，减少空闲 tick 的能力查询。
- Curios 查询异常不再静默吞掉所有 `Throwable`，改为记录一次警告并安全禁用当次饰品效果。
- 公共 API 的 Curios 安装状态查询改用 Forge `ModList`，避免 API 包直接依赖兼容实现。
- “赛博装备”进阶改为获得任意一件饰品即可完成，并同步中英文描述；生命支持箱副作用消减说明与实际默认约 3 倍消退速度保持一致。
