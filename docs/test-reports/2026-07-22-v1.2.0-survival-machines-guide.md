# v1.2.0 生存模式、机器链路与指导书验证报告

日期：2026-07-22（2026-07-23 追加 S-02 闪烁与正面效果图标验证）

## 结论

自动化门禁通过。v1.2.0 的拼接概率与双生产出、种子交易、连续培养速度、培养槽旧瓶迁移、冷凝器库存兼容、血清返瓶、Patchouli 可选指导书和神经过载图标均有确定性覆盖。后续追加的血清效果重平衡保留了等级与时长完整叠加，并覆盖三类主效果、来源专属神经过载、牛奶清除边界与无可选依赖启动。S-02 夜视与失明的短周期闪烁已修复，三张正面效果图标已按 18×18 大块主体重新设计。客户端可进入主菜单并加载全部效果资源；专用服务器可正常启动。

- 三张正面效果贴图均存在、尺寸为 18×18 且文件内容互不相同。
