# v1.2.0 红石与工业兼容合并测试报告（2026-07-23）

## 范围

在 `fix/multiplayer-stability` 的 v1.2.0 基线上审查并合入
`feature/v1.1.7-redstone-industrial`，保留当前机器、GUI、JEI 与存档兼容行为：

- 四台机器接入 `IGNORE/HIGH/LOW` 红石控制与统一比较器语义。
- 四台机器提供分面 Forge `IItemHandler`，并发布材料及机器输入语义标签。
- 培养箱保持 5 槽，底面仅输出成熟产物。
- 冷凝机保持玻璃瓶输入、16 瓶正常库存上限和自动向下传输。
- 不恢复已经移除的冷凝机、拼接机及灌装机 GUI 文字与多余动画。

## 审查修正

- 更新旧能力矩阵测试：培养箱底面由两个历史输出槽改为当前单一成熟产物槽。
- 更新冷凝机能力矩阵：顶部输入玻璃瓶，水平面输入玻璃瓶或输出纯净水，底面仅输出纯净水。
- 删除旧冷凝机暂停与自动注入按钮测试，不恢复已删除的 GUI 功能。
- 修正文档中的 6 槽培养箱、32 瓶冷凝库存和手动下传开关等历史描述。
- 为功能分支历史计划与测试报告增加当前状态说明，保留原始历史结论。
- 清理无效导入、过时的 PULSE 候选说明和合并产生的空白格式问题。
