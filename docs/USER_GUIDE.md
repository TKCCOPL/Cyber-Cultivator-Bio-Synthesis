# Cyber-Cultivator: Bio-Synthesis — 使用指南

## 模组简介

赛博农夫 (Cyber-Cultivator) 是一个中型 Minecraft Forge 模组，用精密的实验室设备取代传统农业，融合遗传育种算法与生物强化血清系统。你需要通过挖掘矿物、种植生化作物、培育基因、合成血清来强化自身能力。

**前置依赖：** Curios API（饰品系统）

---

## 快速开始

### 第一步：挖掘基础矿物

进入地下矿洞，找到两种模组矿石：

| 矿石 | 石镐挖掘 | 掉落物 | 用途 |
|------|---------|--------|------|
| 硅晶矿 (Silicon Ore) | 需要石镐 | 硅碎片 | 数据信号注入、机器合成 |
| 稀土矿 (Rare Earth Ore) | 需要石镐 | 稀土粉末 | 机器核心、高级配方 |

### 第二步：种植基础作物

在创造模式物品栏「赛博农夫」标签页中找到三种种子：

| 作物 | 收获物 | 用途 |
|------|--------|------|
| 纤维草 (Fiber Reed) | 植物纤维 | 合成支持箱 |
| 蛋白质豆 (Protein Soy) | 生化原液 | 所有血清的基础溶剂 |
| 酒精花 (Alcohol Bloom) | 工业乙醇 | S-03 血清配方 |

> 作物种植在耕地上，与原版小麦类似，但最终产物需在培养槽中精耕。

## 核心系统详解

### 大气冷凝器

放置后自动运行，每 30 秒产出 1 个「纯净水瓶」，库存上限 32 个。

- **自动传输：** 将冷凝器放置在培养槽正上方，冷凝器会自动向下注入纯净度 +20（消耗 1 瓶纯净水）
- **漏斗兼容：** 侧面连接漏斗可抽取纯净水瓶
- **红石信号：** 有库存时输出红石信号 15

### 生物培养槽

培养槽是模组的核心耕作方块。放入基因种子后，需要维持三项数值才能让作物生长：

| 数值 | 作用 | 注入方式 | 衰减速度 |
|------|------|---------|---------|
| 营养度 (N) | 低于阈值停止生长 | 生化原液 +25 | 每秒 -1 |
| 纯净度 (P) | 影响作物品质 | 水桶 +20 或冷凝器自动 | 每 2 秒 -1 |
| 数据信号 (D) | 高级作物必需 | 硅碎片 +15 | 每 3 秒 -1 |

**交互操作：**

| 操作 | 方法 |
|------|------|
| 放入种子 | 手持基因种子右键 |
| 注入纯净度 | 手持水桶右键（消耗水桶→空桶） |
| 注入营养度 | 手持生化原液右键 |
| 注入数据信号 | 手持硅碎片右键 |
| 取回种子 | 潜行 + 右键 |
| 查看状态 | 右键（状态栏显示 N/P/D 数值） |

**生长条件：** N > 10 且 P > 10 且 D > 0 时，作物持续生长。进度满 200 后消耗额外营养和纯净度进入下一轮循环。

> 提示：佩戴光谱单片镜并对准培养槽，屏幕左上角会显示 N/P/D 进度条。

### 基因拼接机

将两颗基因种子放入拼接机，自动融合产出新种子。每颗种子携带三个基因属性（1-10）：

- **Gene_Speed** — 生长速度
- **Gene_Yield** — 产量
- **Gene_Potency** — 效价（影响血清品质）

**拼接公式：**
```
新值 = floor((父本 + 母本) / 2) + 随机变异(-2 ~ +2)
结果 clamp 到 1-10 范围
```

**操作方法：**

| 操作 | 方法 |
|------|------|
| 放入种子 | 手持基因种子右键（先放 A，再放 B） |
| 取出结果 | 空手右键（或潜行右键） |
| 取回输入 | 潜行右键（取出最后放入的种子） |

> 策略提示：多次迭代育种可逐步提升基因数值，但变异范围 ±2 意味着每次都有退化风险。建议保留优质亲本。

### 血清灌装机

将突触神经莓加工为高级血清。3 个输入槽 + 1 个输出槽。

**配方：**

| 配方 | 输入材料 | 输出 |
|------|---------|------|
| 莓合成 | 植物纤维 + 工业乙醇 + 生化原液 | 突触神经莓 |
| S-01 突触超频 | 莓 + 生化原液 + 玻璃瓶 | S-01 血清 |
| S-02 视觉强化 | 莓 + 稀土粉末 + 玻璃瓶 | S-02 血清 |
| S-03 代谢加速 | 莓 + 工业乙醇 + 玻璃瓶 | S-03 血清 |

- 加工时间：15 秒（300 tick）
- 漏斗兼容：顶部/侧面注入材料，底部抽取输出
- 交互：右键放入材料，空手/潜行取出输出

## 血清系统

### S-01 突触超频血清

**合成方式：** 血清灌装机 = 突触神经莓 + 生化原液 + 玻璃瓶

**效果：**
- 攻击速度 +25%
- 移动速度 +15%
- 持续获得急迫 II（挖掘速度提升）
- 持续时间：25 秒

**副作用：** 药效结束后进入「神经过载」状态 — 减速 + 饥饿（持续 12+ 秒）

### S-02 视觉强化血清

**合成方式：** 血清灌装机 = 莓 + 稀土粉末 + 玻璃瓶

**效果：**
- 夜视效果
- 32 格范围内所有生物显示发光轮廓（可透视墙壁）
- 持续时间：30 秒

**副作用：** 药效结束后进入「神经过载」状态（持续 15+ 秒）

### S-03 代谢加速血清

**合成方式：** 血清灌装机 = 莓 + 工业乙醇 + 玻璃瓶

**效果：**
- 每 0.5 秒恢复 1+ 点生命值（随 amplifier 增加）
- 急迫效果刷新
- 持续时间：15 秒

**副作用：** 药效结束后进入「神经过载」状态（持续 15+ 秒）

### 副作用缓解

「神经过载」会施加减速 II + 饥饿。可通过以下方式缓解：
- 佩戴「生命支持箱」饰品 — 每 10 秒减少副作用持续时间
- 等待自然消退

## Curios 饰品系统

模组提供 3 件 Curios 饰品，需安装 Curios API 才能使用。

### 光谱单片镜 (Spectrum Monocle)

- **槽位：** 头部
- **功能：** 佩戴后准星对准以下机器时，屏幕左上角显示对应 HUD 信息：
  - **培养槽：** N/P/D 进度条 + 种子状态 + 生长进度 (G) + 预计成熟时间 (ETA)
  - **血清灌装机：** 配方名 + 加工进度条 + 突触活性值
  - **大气冷凝器：** 生产进度条 + 库存量 + 状态（生产中/满/空闲）
  - **基因拼接机：** 父本 A/B 基因数值 + 输出结果基因
  - 查看基因种子时显示隐藏的基因数值
- **合成：** 创造模式获取（当前版本无合成配方）

### 生化脉冲腰带 (Bio-Pulse Belt)

- **槽位：** 腰部
- **功能：** 每秒自动扫描范围内培养槽，消耗背包材料注入数值
  - 营养度 < 阈值 → 消耗背包中生化原液 +25
  - 纯净度 < 阈值 → 消耗背包中纯净水瓶 +20
  - 数据信号 < 阈值 → 消耗背包中硅碎片 +15
- **合成：**
  ```
  铁  硅碎片 铁
  铁  生化原液 铁
     硅碎片
  ```
- **可配置：** 扫描范围（默认 3 格）、各项注入阈值

### 生命支持箱 (Life Support Pack)

- **槽位：** 背部
- **功能：**
  - 持续缓解神经过载副作用（每 10 秒减少副作用时长）
  - 低血量应急：HP ≤ 6 时自动消耗背包中生化原液，回复 4HP + 急迫效果（60 秒冷却）
- **合成：**
  ```
  铁  皮革  铁
  铁  稀土  铁
     植物纤维
  ```

## 突触神经莓

突触神经莓是制造所有高级血清的核心中间产物。

**合成：** 血清灌装机 = 植物纤维 + 工业乙醇 + 生化原液

同时也是食物（回复 2 饱食度），但主要用途是作为血清配方的原料。

## 品质链路系统

原料和血清具有品质属性，从种子基因继承并通过加工链传递。

### 原料品质 NBT

培养槽中的作物成熟时，产出的原料会根据种子的 **Gene_Potency** 写入品质 NBT：

| 原料 | NBT 标签 | 来源 |
|------|---------|------|
| 植物纤维 | Potency | 纤维草种子的 Potency 基因 |
| 工业乙醇 | Purity | 酒精花种子的 Potency 基因 |
| 生化原液 | Concentration | 蛋白质豆种子的 Potency 基因 |

### 突触活性 (Synaptic Activity)

血清灌装机在合成莓或血清时，根据三种原料的品质计算**突触活性**：

```
Activity = round(Potency × 0.25 + Purity × 0.375 + Concentration × 0.375)
结果 clamp 到 1-10 范围
```

活性值越高，血清效果越强。无品质 NBT 的原料默认品质为 5。

### 血清效果缩放

血清的实际效果按 Activity 缩放：
- **持续时间倍率：** `0.5 + Activity × 0.1`（Activity 1 = 0.6x，Activity 10 = 1.5x）
- **起始等级：** Activity ≥ 8 时起步 II 级，否则起步 I 级

---

## 血清叠加升级机制

多次饮用同种血清可叠加效果：

| 机制 | 规则 |
|------|------|
| 等级叠加 | 每次饮用 amplifier +1，上限 V 级（amplifier 4） |
| 持续时间累加 | 新血清持续时间 + 已有效果剩余时间，上限 5 分钟 |
| 高品质起步 | Activity ≥ 8 的血清首次饮用即为 II 级 |
| 副作用增强 | 等级越高，药效结束后的神经过载越严重 |

## 配置选项

模组安装后会在 `config/` 目录生成配置文件，可调整以下参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| beltScanRange | 3 | 腰带扫描培养槽的范围（格） |
| beltNutritionThreshold | 50 | 营养度低于此值时腰带自动注入 |
| beltPurityThreshold | 50 | 纯净度低于此值时腰带自动注入 |
| beltDataSignalThreshold | 25 | 数据信号低于此值时腰带自动注入 |
| packEffectReductionRate | 2 | 支持箱每 10 秒减少副作用的时长（tick） |
| packHealThreshold | 6.0 | 支持箱触发应急治疗的血量阈值 |
| packHealCooldown | 1200 | 应急治疗冷却时间（tick，默认 60 秒） |
| monocleHudRange | 8 | 单片镜 HUD 显示的最大距离（格） |

## 自动化产线布局

```
[大气冷凝器]        ← 每 30 秒自动产出纯净水
      ↓ 自动注入纯净度
[生物培养槽]        ← 放入种子，腰带自动注入营养/信号
      ↓ 作物成熟后取出原料
[血清灌装机]        ← 原料合成莓 → 莓+辅料+瓶 → 血清
      ↓ 输出
[突触神经莓 / S-01 / S-02 / S-03 血清]
```

漏斗可连接在冷凝器侧面抽取纯净水，灌装机顶部/侧面注入材料、底部抽取输出。

## 常见问题

**Q: 作物一直不生长？**
A: 检查培养槽的 N/P/D 数值。需要 N > 10 且 P > 10 且 D > 0 才能生长。潜行右键或佩戴单片镜查看状态。

**Q: 基因数值怎么看？**
A: 佩戴光谱单片镜后，将鼠标悬停在基因种子上即可查看 Gene_Speed / Gene_Yield / Gene_Potency 数值。未佩戴时显示「需要佩戴光谱单片镜以解析基因」。

**Q: 血清副作用太强怎么办？**
A: 佩戴生命支持箱可加速副作用消退。或在安全环境下使用血清，等待副作用自然结束。

**Q: 没有 Curios API 能用饰品吗？**
A: 不能。饰品功能依赖 Curios API。但血清系统、培养槽、拼接机等核心功能不受影响。

**Q: 矿石不生成？**
A: 硅晶矿和稀土矿使用 Forge biome modifier 生成，正常创建世界即可找到。
