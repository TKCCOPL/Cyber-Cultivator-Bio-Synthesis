## 模组简介

用精密实验室设备取代传统农业。挖掘矿物、培育基因作物、合成血清，通过基因优化和药剂注射获得超凡能力。每颗种子都携带独特的基因密码，每一次育种都是一场赌博。

**核心循环：** 挖矿 → 种植 → 培养槽精耕 → 基因育种 → 血清合成 → 强化自身

## 核心设施

### 大气冷凝器
自动从空气中凝结纯净水，每 30 秒产出 1 瓶，库存上限 32。放置在培养槽正上方可自动注入纯净度。支持漏斗抽取。

### 生物培养槽
核心耕作方块。放入基因种子后，需维持三项数值才能生长：

| 数值 | 注入方式 | 作用 |
|------|---------|------|
| 营养度 | 生化原液 | 低于阈值停止生长 |
| 纯净度 | 纯净水瓶 / 冷凝器自动 | 影响作物品质 |
| 数据信号 | 硅碎片 | 高级作物必需 |

### 基因拼接机
放入两颗基因种子，自动融合产出新种子。每颗种子携带三个基因属性（1-10）：
- **Gene_Speed** — 生长速度
- **Gene_Yield** — 产量
- **Gene_Potency** — 效价（影响血清品质）

**拼接公式：** `新值 = floor((父本 + 母本) / 2) + 随机变异(-2 ~ +2)`

### 血清灌装机
将突触神经莓加工为高级血清。支持漏斗自动化。单片镜准星对准可显示当前配方、加工进度和突触活性值。

## 强化血清

通过基因育种提升 `Gene_Potency`，制造效果更强的血清。血清支持叠加升级（最高 V 级），每次饮用提升一级效果并刷新持续时间。

### 品质链路

```
种子 Potency → 原料品质 NBT → 灌装机合成莓（突触活性）→ 血清（继承活性）→ 饮用效果按活性缩放
```

三种原料携带不同品质标签：植物纤维(Potency)、工业乙醇(Purity)、生化原液(Concentration)。突触活性 = 加权平均，范围 1-10。

### 血清列表

| 血清 | 效果 | 基础时长 | 副作用 |
|------|------|---------|--------|
| **S-01 突触超频** | 攻速+力量（随等级增长）+ 抗性（上限 III） | 25 秒 | 神经过载：凋零 + 饥饿 |
| **S-02 视觉强化** | 夜视 + 发光范围 16-48 格（随等级增长）+ 抗火（上限 III） | 30 秒 | 神经过载：失明 + 饥饿 |
| **S-03 代谢加速** | 持续回血 + 移速+跳跃提升（上限 III） | 15 秒 | 神经过载：缓慢 + 中毒 |

### 叠加机制

- 多次饮用同种血清：amplifier +1（上限 VIII 级），持续时间累加（上限 5 分钟）
- 高品质血清（活性 ≥ 8）起步 amplifier = 1（II 级）
- 副作用随等级同步增强：等级越高，神经过载持续时间越长、强度越大
- 副作用差异化：S-01 凋零+饥饿、S-02 失明+饥饿、S-03 缓慢+中毒
- 佩戴生命支持箱可加速副作用消退

## Curios 饰品

通过探索战利品箱获得（末地城、废弃矿坑、地牢等）：

| 饰品 | 槽位 | 功能 |
|------|------|------|
| 光谱单片镜 | 头部 | 准星对准机器时显示 HUD：培养槽(N/P/D/生长/ETA)、灌装机(配方/进度/活性)、冷凝器(进度/库存)、拼接机(种子基因/结果) |
| 生化脉冲腰带 | 腰部 | 自动扫描范围内培养槽，消耗背包材料注入三项数值 |
| 生命支持箱 | 背部 | 加速神经过载消退，低血量时自动注射治疗 |

## 自动化产线

```
[大气冷凝器] → 纯净水自动注入
      ↓
[生物培养槽] ← 腰带自动注入营养/信号
      ↓ 作物成熟
[血清灌装机] → 高级血清
```

漏斗可连接在冷凝器侧面抽取纯净水，灌装机顶部/侧面注入材料、底部抽取输出。
