# 模组开发企划书：赛博农夫 (Cyber-Cultivator: Bio-Synthesis)

## 📌 核心概述
* **模组定位：** 中型体量 (Growth)，基于 Forge 框架开发。
* **核心玩法：** 写实环境挑战 (硬核数值) + 遗传育种算法 + 生物强化血清。
* **美术风格：** 赛博朋克/实验室风。标准 **16x16 像素**，采用扁平化、高对比度配色，强调边缘霓虹高光与半透明玻璃质感。
* **依赖前置：** Curios API (核心饰品交互)。

---

## 一、 独立资源体系 (闭环生态)

为了构建完整的工业流水线，模组提供独立的底层物料：

### 1. 基础矿物
* **硅晶矿 (Silicon Ore)：** 掉落“硅碎片”。用于合成电路板、传感器和培养槽玻璃。
* **稀土矿 (Rare Earth Ore)：** 掉落“稀土粉末”。用于制作精密机器核心和激光器。

### 2. 基础作物 (生化耗材)
| 作物名称 | 16x16 视觉特征 | 核心用途 |
| :--- | :--- | :--- |
| **纤维草 (Fiber Reed)** | 翠绿色，笔直似玻璃纤维，带金属光泽。 | 提取**植物纤维**，合成实验室防护服或过滤膜。 |
| **蛋白质豆 (Protein Soy)**| 灰白色方形豆荚，呈阵列状。 | 压榨获得**生化原液**，所有血清的基础溶剂。 |
| **酒精花 (Alcohol Bloom)**| 半透明淡紫花瓣，亮红色花蕊。 | 提炼**工业乙醇**，作机器清洁剂或燃料。 |

---

## 二、 核心设施与多方块机器

传统的农田被替换为精密的实验室仪器，所有机器均需特定的运行条件：

1. **大气冷凝器 (Atmospheric Condenser)**
   * **功能：** 消耗能量从空气提取“纯净水”。
   * **外观：** 顶部带旋转风扇，侧面有半透明水管。
2. **生物培养槽 (Bio-Incubator)**
   * **功能：** **核心耕作方块**。需持续注入纯净水和营养液。
   * **外观：** 三面玻璃，内部填充发光液体，底部为金属基座。
3. **基因拼接机 (Gene Splicer)**
   * **功能：** 根据遗传算法，消耗两颗基础种子合成变异新种子。
   * **外观：** 两个水平物品槽，中心悬浮垂直激光发射点。
4. **血清灌装机 (Serum Bottler)**
   * **功能：** 将成熟的高级作物精华提取并浓缩为“强化血清”。
   * **外观：** 向下的机械针头，底部放置空胶囊底座。

## 三、 硬核生长与遗传育种逻辑

### 1. 动态生长环境 (TileEntity 数据)
作物生长不再依赖骨粉，而是受培养槽内三个动态数值控制：
* **营养度 (Nutrition)：** 维持在 20%-100% 之间，过低则停止生长，0% 则枯萎。
* **纯净度 (Purity)：** 必须使用纯净水，杂质超标会导致作物变异为废料。
* **数据信号 (Data Signal)：** 培养槽半径 5 格内需存在“数据服务器”方块，为高级作物提供运行环境。

### 2. 基因继承机制 (NBT 标签)
每颗种子自带隐藏的基因代码：`{Gene_Speed: 1-10, Gene_Yield: 1-10, Gene_Potency: 1-10}`。
* **合成公式：** `新种子数值 = (父本 + 母本) / 2 + 随机变异浮动 (-1 到 +2)`。

## 四、 Curios 饰品交互系统

饰品是进行农业操作的刚需工具，与 Curios API 深度绑定：

| 饰品名称 | Curios 槽位 | 核心功能 |
| :--- | :--- | :--- |
| **光谱单片镜 (Spectrum Monocle)** | Head (头部) | **UI 解锁：** 佩戴后可透视培养槽的精准养分数据及种子的 NBT 基因等级。 |
| **生化脉冲腰带 (Bio-Pulse Belt)** | Belt (腰部) | **自动化：** 自动消耗玩家背包内的营养液，为半径 3 格内的培养槽充能。 |
| **生命支持箱 (Life Support Pack)**| Back (背部) | **维稳：** 减少高阶血清带来的副作用，生命值极低时自动注射治疗药剂。 |

---

## 五、 明星产出：强化血清 (Enhancements)

通过不断迭代 `Gene_Potency` (效价)，玩家可制造出效果极强的强化药剂：

* **核心作物：突触神经莓 (Synaptic Neural-Berry)**
  * 半透明荧光蓝茎秆，内部具神经元纹理。
* **衍生血清效果：**
  * **S-01 突触超频：** 大幅提升挖掘速度与攻击速度。
  * **S-02 视觉强化：** 永久夜视，透视墙后生物轮廓。
  * **S-03 代谢加速：** 瞬间高额回血，赋予短时急迫效果。
* **赛博代价 (副作用)：** 药效结束后陷入“神经过载”状态（减速/饥饿）。使用更高等级基因培育的血清可降低此副作用。
